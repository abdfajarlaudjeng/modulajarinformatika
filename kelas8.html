Modul Informatika 8 & 9

Kelas 8: Hardware (Perangkat Keras)

Materi perangkat keras di kelas 8 memperdalam pemahaman tentang fungsi tiap komponen.
Contoh: Motherboard, RAM, Harddisk, SSD, GPU, Peripheral (printer, scanner).

Gambar: Komponen hardware pada komputer modern
Aktivitas: Amati komputer di kelas/lab, lalu buat skema dan fungsi singkat tiap komponen hardware yang kamu temukan!

Kelas 8: Software (Perangkat Lunak)

Software dibagi menjadi sistem operasi, program aplikasi, dan utility.
Contoh aplikasi: Spreadsheet (Excel), desain grafis (GIMP), pemrograman (Scratch, Python).

Gambar: Ikon aplikasi perangkat lunak
Aktivitas: Praktikkan membuat dokumen di aplikasi pengolah kata, lalu simpan dan buka kembali hasilnya!

Kelas 8: Brainware

Di kelas 8, brainware tidak hanya pengguna biasa, tapi juga pengembang dan administrator.
Contoh: Guru informatika, admin lab komputer, siswa yang membuat program sederhana.

Gambar: Brainware di sekolah
Diskusi: Apa peran admin komputer di sekolah? Bagaimana cara menjaga keamanan data di komputer?

Kelas 8: Sistem Operasi

Kelas 8 mempelajari fitur-fitur sistem operasi:
Manajemen file, instalasi aplikasi, multi-user, keamanan (password), pengaturan jaringan.

Gambar: Salah satu sistem operasi populer (Ubuntu)
Praktik: Coba buat folder, rename, copy, dan hapus file di komputer sekolah secara langsung!

Kelas 9: Hardware (Perangkat Keras)

Di kelas 9, hardware dikaitkan dengan perkembangan teknologi terbaru.
Bahas juga IoT (Internet of Things), perangkat mobile, dan hardware jaringan (router, switch).

Gambar: Perangkat keras jaringan dan IoT
Aktivitas: Telusuri, apa itu perangkat IoT? Cari contoh perangkat IoT di rumah/sekolahmu!

Kelas 9: Software (Perangkat Lunak)

Di kelas 9, bahas software berbasis web dan mobile.
Contoh: Aplikasi berbasis cloud (Google Docs), aplikasi Android/iOS, software keamanan (antivirus).

Gambar: Logo aplikasi dan software keamanan
Tugas: Instal aplikasi produktivitas di ponselmu, lalu bandingkan fiturnya dengan aplikasi desktop!

Kelas 9: Brainware

Brainware berkembang menjadi pengembang aplikasi, admin IT, dan pengguna digital cerdas.
Diskusi etika digital, keamanan data, serta peran brainware di era digital.

Gambar: Pengembang dan admin IT
Diskusi: Mengapa brainware harus memahami keamanan data dan etika penggunaan teknologi?

Kelas 9: Sistem Operasi

Bahas sistem operasi pada berbagai perangkat: komputer, smartphone, IoT.
Perbandingan OS: Windows vs Linux vs Android vs iOS. Fitur keamanan, update, dan open source.

Gambar: Logo sistem operasi mobile (Android)
Aktivitas: Identifikasi sistem operasi pada gadget yang kamu miliki. Apa kelebihan OS tersebut?
